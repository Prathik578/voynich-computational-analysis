# Voynich Computational Analysis

A research project exploring the Voynich Manuscript through computational and statistical analysis.

## Overview

The Voynich Manuscript is one of the most mysterious texts ever discovered. Despite decades of study, its language and meaning remain unknown.

This project investigates the manuscript using data-driven methods to identify statistical patterns and compare them with known languages.

## Research Goals

* Character frequency analysis
* Entropy analysis
* N-gram analysis
* Language comparison
* Pattern visualization

## Technologies

* Python
* NumPy
